                        function getValues() {
                        //get values from the page
                        let startValue = document.getElementById("startValue").value;
                        let endValue = document.getElementById("endValue").value;

                        //We need to validate our input
                        //parse into Integers
                        startValue = parseInt(startValue);
                        endValue = parseInt(endValue);

                        if (Number.isInteger(startValue) && Number.isInteger(endValue)) {
                            //we call generateNumbers
                            let numbers = generateNumbers(startValue, endValue);
                            //we call displayNumbers
                            displayNumbers(numbers);
                            } else {
                        alert("You must use Integers only")
                            }
                        }

                        //generate numbers from the startValue to the endValue
                        //logic function(s)
                        function generateNumbers(sValue, eValue) {
                        let numbers = [];

                        //we want to get all numbers from start to end

                        for (let i = sValue; i <= eValue; i++) {
                            //this will execute in a loop until index=eValue
                            numbers.push(i);
                            }
                            return numbers;
                        }

                        function displayNumbers(numbers) {
                            let templateRows="" ;
                            for (let i=0; i < numbers.length; i++) {
                            let className="even" ;
                            let number=numbers[i];
                                if (number % 2==0) {
                                    className="even" ;
                                }else{ className="odd" ;
                                }
                                //This does render correctly with Prism, see the source.
                                templateRows += `<tr><td class="${className}">${number}</td></tr>`;
                            }

                        document.getElementById("results").innerHTML = templateRows;

                        }

Three functions are the engine that drives this code:

getValues

getValues accepts(gets) the user's input from the page. The info is then accessed by getElementById to pull the values from the code. The code then passes those values to the generateNumbers function. That function returns an array of values and passes it on to the displayNumbers function.

generateNumbers

generateNumbers consists of two arguments of startValue and endValue. I created a varaible called numbers so that held an array so the for loop could generate all numbers between startValue and endValue.

displayNumbers

displayNumbers uses a single argument, numbers. The variable numbers is an array that holds the value of startValue and endValue. Thus created a varaible className that is linked to the CSS class I used to BOLD the numbers. I also created templateRows to hold the HTML that is written to the page.

A for loop was created to confirm whether or not the numbers are "odd" or "even". This made good use of the modulo (%) operator, so long as the number is divisible by two. If it returned zero (0), then it is in fact an even number. The correct className is then pushed to the HTML row to be displayed. Each iteration of the loop is added to the templateRows variable and then added to the respectec page table.
